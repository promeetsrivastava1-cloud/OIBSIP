# 📊 Retail Sales Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project focuses on **Exploratory Data Analysis (EDA)** of a retail sales dataset using Python. The objective is to uncover meaningful business insights by analyzing customer demographics, purchasing behavior, product performance, and sales trends through data visualization and statistical analysis.

The project demonstrates essential data analytics skills, including data cleaning, exploratory analysis, visualization, and business insight generation.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Explore sales performance across product categories.
- Identify monthly and quarterly sales trends.
- Discover top-performing products and regions.
- Generate actionable business recommendations based on data.

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset Information

The dataset contains retail sales transaction records with the following attributes:

- Transaction ID
- Date
- Customer Age
- Gender
- Product Category
- Quantity Purchased
- Price Per Unit
- Total Amount

### 1️⃣ Data Understanding
- Loaded the dataset
- Examined dataset structure
- Checked data types
- Explored descriptive statistics

### 2️⃣ Data Cleaning
- Checked for missing values
- Verified data consistency
- Converted date columns into datetime format
- Prepared the dataset for analysis

### 3️⃣ Exploratory Data Analysis (EDA)

The following analyses were performed:

- Dataset Overview
- Statistical Summary
- Customer Age Distribution
- Gender Distribution
- Product Category Analysis
- Monthly Sales Trend
- Quarterly Sales Trend
- Regional Sales Analysis
- Top 10 Selling Products
- Discount vs Sales Analysis
- Correlation Heatmap

## 📈 Key Insights

- Product categories contributed differently to overall sales performance.
- Monthly and quarterly analysis revealed seasonal sales trends.
- Customer demographics influenced purchasing behavior.
- Certain products consistently generated higher revenue.
- Regional performance varied significantly.
- Correlation analysis highlighted relationships among numerical features.
